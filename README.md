# mygit

A minimal command-line tool example using Python and argparse.

## Features

- `init` command: Prints "Hello World!!" to the console.

## Installation

From the project root, install in editable mode:
```bash
pip install -e .
```

## Usage

Run the CLI with:
```bash
mygit init
```

## Project Structure

```
mygit/
    cli.py
setup.py
README.md
```

## Requirements

- Python 3.7+
